# DrawnCurve SVG / Figma handoff

Files:
- 01_utility_panel.svg
- 02_circular_time.svg
- 03_timeline_clip.svg
- 04_scientific_plot.svg
- drawncurve_wireframes_board.svg

These SVGs are plain vector shapes and text, suitable for import into Figma.

Recommended Figma workflow:
1. Create a new file.
2. Import each SVG as a frame reference.
3. Keep repeated controls editable at first.
4. Replace text with Figma styles:
   - Title / 28 / Semibold
   - Label / 14 / Semibold
   - Small / 12 / Regular
5. Convert repeated controls into components:
   - button
   - active button
   - slider
   - dual-thumb range slider
   - panel
6. Prototype the main curve surface as one hotspot first.

Intent:
- non-skeuomorphic
- performance-first
- minimal visible toolkit while drawing
- mapping deferred
